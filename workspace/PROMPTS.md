# PROMPTS.md - 实用 Prompt 模板

云可以随时使用这些 prompt 来帮助 Claw 更好地工作。

---

## 📂 项目管理

### 创建新项目

```
帮我创建一个新项目，名字叫 [项目名]，我想做一个 [描述功能]
```

### 查看项目状态

```
帮我看看 [项目名] 的进度，有什么需要处理的？
```

---

## 🔍 信息查询

### 搜索网页

```
帮我搜索 [关键词]，我想了解 [具体想了解什么]
```

### 读取网页

```
帮我看看这个网页的内容：[URL]
```

---

## 📝 文档和笔记

### 写笔记

```
帮我记一下：[内容]
```

### 整理记忆

```
帮我把最近的记忆整理一下，提交到 GitHub
```

---

## 🛠️ 开发相关

### 写代码

```
帮我写一个 [语言] 程序，功能是 [描述]
```

### 调试

```
这段代码有问题，帮我看看：[代码]
```

### GitHub 操作

```
帮我在 GitHub 上创建一个 issue/pr，内容是 [描述]
```

---

## 💬 日常对话

### 闲聊

```
随便聊聊吧，你想聊什么？
```

### 讲故事

```
给我讲一个关于 [主题] 的故事
```

### 学习

```
我想学习 [主题]，你能教我吗？从基础开始
```

---

## ⚙️ 系统操作

### 唤醒 Claw

```
（在 GitHub Actions 里点 Mobile-One-Click-Start）
```

### 检查状态

```
帮我看看 OpenClaw 的运行状态
```

### 保存记忆

```
帮我保存今天的记忆到 GitHub
```

---

## 🎯 特殊功能

### 深度推理

```
仔细思考一下这个问题：[问题]
```

### 多步骤任务

```
我需要完成一个复杂任务：[描述]
请帮我规划步骤并逐步执行
```

### 代码审查

```
帮我审查这段代码，看看有没有问题：[代码]
```

---

## 💡 使用技巧

1. **越具体越好**：告诉 Claw 你想做什么，不要含糊
2. **可以追问**：如果回答不够详细，可以追问
3. **可以纠正**：如果 Claw 理解错了，直接纠正
4. **可以提要求**：比如"用简单的话解释"、"给我举例"
